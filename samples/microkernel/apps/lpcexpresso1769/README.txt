Title: Synchronisation

Description:

A simple Hello World example using the microkernel.

--------------------------------------------------------------------------------

Building and Running Project:

This microkernel project outputs to the console.  It can be built and executed
on an lpc1768 board as follows:

    make 

--------------------------------------------------------------------------------

Troubleshooting:

Check that you're using the correct pin mux configuration or the correct uart - this
project uses UART3 on port 0.25 and 0.26

--------------------------------------------------------------------------------

Sample Output:

Hello World!
